# 道语言原语规范（ABI 契约）
> 版本 v1.0 | 2026-06-04
> 本文定义道语言核心原语的行为契约。
> 任何运行时实现（Python、C、Rust、Zig…）必须遵守此规范，
> 以保证 442 个 .ku thought 无需修改即可运行。

---

## 零、契约原则

1. **实现无关** — 原语行为由本文档定义，不由 runtime.py 的实现定义
2. **语义等价** — 所有实现必须对同一组输入产生完全相同的输出
3. **错误处理** — 参数不合法时返回 `Error: <描述>` 字符串，不抛异常
4. **不可变参数** — 原语不修改传入的参数对象

---

## 一、核心 I/O 原语

### 示
```
示(x)
  → x 任意类型
  → 返回 x 本身
  行为：将 x 的字符串表示输出到标准输出
  边界：None → 输出空行；含中文 → UTF-8 编码输出
```

### 读文件
```
读文件(path)
  path: 字符串（绝对路径或相对路径）
  → 文件内容的字符串；失败返回 Error: <原因>
  行为：以 UTF-8 读取文件全部内容
  边界：文件不存在 → "Error: file not found: <path>"
```

### 写文件
```
写文件(path, content)
  path: 字符串
  content: 任意类型（自动转字符串）
  → 成功返回 True；失败返回 Error: <原因>
  行为：以 UTF-8 写入文件（覆盖写）
  边界：目录不存在 → 自动创建父目录
```

## 二、字符串原语

### 文_分割
```
文_分割(s, delimiter)
  s: 字符串
  delimiter: 字符串
  → 字符串列表
  行为：按 delimiter 分割 s（等同于 Python str.split）
  边界：空字符串 → [""]；分隔符不在 s 中 → [s]
```

### 文_包含
```
文_包含(s, substr)
  s: 字符串
  substr: 字符串
  → 布尔值（True/False）
  行为：检查 substr 是否为 s 的子串
  边界：substr 为空 → True
```

### 文_替换
```
文_替换(s, old, new)
  s, old, new: 字符串
  → 字符串
  行为：将 s 中所有 old 替换为 new（等同于 str.replace）
```

### 文_去空白
```
文_去空白(s)
  s: 字符串
  → 字符串
  行为：去除 s 首尾的空白字符（空格、换行、制表符等）
```

### 文_是否为空
```
文_是否为空(s)
  s: 字符串
  → 布尔值
  行为：判断字符串是否为空或仅含空白字符
  边界："" → True；"  " → True；"abc" → False
```

### 文_小写 / 文_大写
```
文_小写(s) → 字符串    # s.lower()
文_大写(s) → 字符串    # s.upper()
```

### 文_起头 / 文_结尾
```
文_起头(s, prefix) → 布尔    # s.startswith(prefix)
文_结尾(s, suffix) → 布尔    # s.endswith(suffix)
```

### 文_格式化
```
文_格式化(template, values)
  template: 字符串（含 {} 占位符）
  values: 列表
  → 字符串
  行为：用 values 依次替换 template 中的 {}
  例：文_格式化("你好{}", ["世界"]) → "你好世界"
```

### 文_转字符串
```
文_转字符串(x)
  x: 任意类型
  → 字符串
  行为：将任意值转为字符串表示
  边界：None → "None"；整数 → 字符串数字；列表 → 字符串形式
```

## 三、类型原语

### 整数 / 浮点数 / 布尔
```
整数(x) → 整数      # int(x)
浮点数(x) → 浮点数   # float(x)
布尔(x) → 布尔      # bool(x)
```

### 类型检查
```
是字符串(x) → 布尔   # isinstance(x, str)
是列表(x) → 布尔     # isinstance(x, list)
是字典(x) → 布尔     # isinstance(x, dict)
是整数(x) → 布尔     # isinstance(x, int)
是空(x) → 布尔       # x is None
```

### 列表长度 / 范围
```
列表长度(x) → 整数    # len(x) 如果实现了 __len__
范围(n) → 整数列表    # list(range(n)) → [0, 1, ..., n-1]
```

### 类型
```
类型(x) → 字符串     # type(x).__name__
```

## 四、字典原语

```
字典键(d) → 键列表        # list(d.keys())
字典值(d) → 值列表        # list(d.values())
合并(a, b) → 字典         # {**a, **b}
含有(obj, key) → 布尔     # key in obj / key in dict / 0 <= key < len(list)
```

---

## 五、基础数据结构原语

```
追加(lst, val) → 列表     # lst.append(val); return lst
切片(obj, start, end) → 同类型
  行为：obj[int(start):int(end)]
有序(ch) → 整数           # ord(ch[0]) 
字符(n) → 字符串          # chr(int(n))
```

## 六、网络原语

```
取(网址) → 字典             # HTTP GET，返回 {"ok":..., "body":..., "status":...}
取JSON(网址) → 字典         # HTTP GET，自动解析 JSON body
发(网址, 内容) → 字典        # HTTP POST
发JSON(网址, 数据) → 字典    # HTTP POST JSON
改(网址, 内容) → 字典        # HTTP PUT
删(网址) → 字典             # HTTP DELETE
```

## 七、认知层原语（agent_core.ku）

### __decompose
```
思 __decompose(goal) {
    根据 goal 文本中的关键词（"refactor"、"test"等）
    返回任务分解列表 [{type, name}, ...]
    默认返回 [{type:"analyze"}, {type:"execute"}]
}
```

### __execute
```
思 __execute(task) {
    匹 task["type"] 为
        "analyze" → __exec_analyze(task)
        "plan" → __exec_plan(task)
        ...
}
```

## 八、原语依赖树

```
核心原语（~40个 Python 实现）
    │
    ├─ 标准库（std/*.ku，~200 个 thought，ku 实现）
    │   ├─ std/string.ku  — 字符串扩展函数（去空白行、连接、重复等）
    │   ├─ std/list.ku     — 列表扩展（映射、过滤、排序等）
    │   ├─ std/math.ku     — 数学扩展（sin, cos, sqrt, 统计等）
    │   ├─ std/fs.ku       — 文件系统扩展（复制、移动、删除）
    │   ├─ std/http.ku     — HTTP 扩展（取、发、改、删）
    │   └─ std/…           — 其他标准库
    │
    ├─ 认知层（~4 个核心文件，ku 实现）
    │   ├─ agent_core.ku    — 任务分解与执行调度
    │   ├─ memory_system.ku — 持久化记忆
    │   ├─ planner.ku       — 规划器
    │   └─ tool_registry.ku — 工具注册表
    │
    └─ 内置函数（_inject_builtins，Python 实现）
        └─ 运算符（+ - * / % == != < > <= >= and or not）
```

## 九、移植清单

要从 Python 移植到 C，需实现以下 40 个原语（按优先级分三组）：

### P0 — 计算内核（7个）
```
+, -, *, /, %, ==, <
```
### P1 — 核心认知原语（8个）
```
示, 读文件, 写文件, 文_分割, 文_包含, 文_替换,
文_去空白, 文_是否为空
```
### P2 — 扩展原语（25个）
```
文_小写, 文_大写, 文_起头, 文_结尾, 文_格式化, 文_转字符串,
整数, 浮点数, 布尔, 是字符串, 是列表, 是字典,
是整数, 是空, 列表长度, 范围, 类型,
字典键, 字典值, 合并, 含有, 追加, 切片, 有序, 字符
```
